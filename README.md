# 🛒 Blinkit Sales Data Analysis — Python Project

A complete end-to-end **Data Analysis project** on Blinkit (India's Last Minute Grocery App) sales data using Python.

---

## 📌 Project Overview

This project analyzes Blinkit's outlet and product sales data to derive actionable business insights. It covers data cleaning, KPI calculation, and multiple chart visualizations to understand sales patterns across product types, outlet sizes, and locations.

---

## 📊 KPIs Analyzed

| KPI | Value |
|-----|-------|
| Total Sales | $1,201,681 |
| Average Sales | $141 |
| No. of Items Sold | 8,523 |
| Average Rating | 4.0 |

---

## 📈 Charts & Visualizations

- **Sales by Fat Content** — Pie chart (Low Fat vs Regular)
- **Total Sales by Item Type** — Bar chart with value labels
- **Fat Content by Outlet Tier** — Grouped bar chart
- **Sales by Outlet Establishment Year** — Line chart with annotations
- **Sales by Outlet Size** — Pie chart
- **Total Sales by Outlet Location Type** — Horizontal bar chart

---

## 🗂️ Dataset

- **Source:** Blinkit Grocery Sales Dataset
- **Size:** 8,523 rows
- **Key Columns:** `Item Fat Content`, `Item Type`, `Outlet Size`, `Outlet Location Type`, `Outlet Establishment Year`, `Sales`, `Rating`

## 🚀 How to Run

1. Clone the repository:
```bash

cd blinkit-analysis
```

2. Install dependencies:
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Open `blinkit_analysis_in_python.ipynb` and run all cells.

## 💡 Key Insights

- **Low Fat** products generate the majority of total sales
- **Tier 3** outlets contribute significantly to overall revenue
- Outlets established around **2018** show the highest sales peak
- **Supermarket Type 1** is the top-performing outlet type
- **Fruits & Vegetables** and **Snack Foods** are top-selling item types
